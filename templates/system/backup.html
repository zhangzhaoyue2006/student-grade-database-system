数据库备份说明

课程项目中提供备份命令说明，便于管理员手动导出数据库。

返回
备份命令
mysqldump --default-character-set=utf8mb4 -u root -p student_grade_system > backup.sql
恢复命令
mysql --default-character-set=utf8mb4 -u root -p student_grade_system < backup.sql
说明
执行命令后输入 MySQL 密码即可完成导出或恢复。备份文件建议按日期命名，例如 backup_2026_05_21.sql。
